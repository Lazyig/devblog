Ubuntu에서 NVM 설치

2022-01-26 20:52:47

우분투 환경에서 NVM 설치 과정에 대해서 간략하게 메모한다.

설치

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

위의 명령어를 통하여 인터넷만 있다면 간편하게 설치를 할 수 있다.

PATH 설정

설치된 NVM을 편리하게 사용하기 위해서는 PATH 설정을 해주어야 한다.

설치시 .bashrc 파일에 자동으로 등록이 되어 있으니 다시 연결하거나 다음 명령어로 스크립트를 재실행 해준다.

source ~/.bashrc

다른 쉘 이용자는 bashrc 파일 내부에 있는 다음 스크립트를 자신의 쉘에 맞는 파일에다가 넣어준다.

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"  # This loads nvm
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"  # This loads nvm bash_completion

zsh사용자는 ~/.zshrc 파일에 넣으면 된다.

그 후에 해당 스크립트를 재 실행해주면 된다.

노드 버전 설치

nvm install 버전

위 명령어로 설치를 하면 된다.

노드 버전 활성화

nvm use 버전

해당 명령어로 해당 버전을 사용하도록 변경한다.
