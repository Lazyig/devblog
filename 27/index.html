NPM 업데이트 하기

2021-10-29 00:04:28

이미 Node.js와 NPM이 설치되어 있고, 이를 업데이트 하기 위한 방법은 간단하다.

그리고 업데이트를 진행하다가 아쉽게도 시행착오가 있었으니 여러분은

  1. Node.js
  2. NPM

순서대로 업데이트 해주기를 바랍니다.

NPM 업데이트

# npm install -g npm

npm으로 npm을 업데이트 시켜주면 된다.

PS C:\Users\User\Desktop> npm -v
6.9.0
PS C:\Users\User\Desktop> npm install -g npm
C:\Users\User\AppData\Roaming\npm\npm -> C:\Users\User\AppData\Roaming\npm\node_modules\npm\bin\npm-cli.js
C:\Users\User\AppData\Roaming\npm\npx -> C:\Users\User\AppData\Roaming\npm\node_modules\npm\bin\npx-cli.js
+ npm@8.1.1
added 218 packages from 96 contributors in 5.057s
PS C:\Users\User\Desktop> npm -v
(node:24144) ExperimentalWarning: The fs.promises API is experimental
npm WARN npm npm does not support Node.js v10.16.0
npm WARN npm You should probably upgrade to a newer version of node as we
npm WARN npm can't make any promises that npm will work with this version.
npm WARN npm You can find the latest version at https://nodejs.org/
8.1.1

다음과 같이 6.9.0에서 8.1.1 버전으로 업데이트가 정상적으로 되었다!

여기에 오류가 조금 있는데. Node.js의 버전이 NPM버전과 호환이 안되는 문제다.

그래서 Node.js를 먼저 업데이트를 했어야했는데. 이미 늦었다.

이럴 경우 처음 설치했던 것 처럼 상위버전을 설치해주면 된다.

저는 윈도우 환경이기 때문에 그냥 다시 받아서 설치했습니다.

PS C:\Users\User\Desktop> node -v
v17.0.1

설치 완료!
