MacOS 특수 키 문자코드

2022-01-11 01:49:11

글을 쓸 때. 맥 OS 유저를 위해서 해당 단축키의 그림을 추가 작성하고 싶을 수 있다.

이에 대한 문자코드와 간단한 입력법을 메모한다.

유니코드

기호 이름 유니코드
⌃ Control U+2303
⌥ Option U+2325
⌘ Command U+2318
⇧ Shift U+21E7
␣ Space U+2423
⌫ Back Space U+232B

대표적으로 자주 쓰이게 되는 조합키 목록

위의 유니코드를 참고하면 된다.

맥에서 입력법

Control + Command + Space (⌃⌘␣)를 누르면 이모지 입력창이 나타나게 된다.

여기 검색에서 위의 검색 창에서 유니코드를 입력해주면 검색이 된다.

`

사용하고 나면 자주 사용하는 항목으로 떠서 편리하게 사용할 수 있다.
