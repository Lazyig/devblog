리눅스 umount 오류 시

2021-10-13 21:55:00

리눅스에서 저장장치를 마운트하고 사용하다가 이를 제거하려고 할 때 오류가 발생하는 경우가 있다.

오류 내용:

umount: 경로: device is busy.

(In some cases useful info about processes that use the device is found by lsof(8) or fuser(1))

다음과 같은 오류가 발생하였다. 이 경우에는 어떤 프로세스가 해당 저장소를 사용하고 있는 경우에 발생하게 된다.

# fuser -ch (마운트 경로)

다음과 같은 명령어로 프로세스를 종료할 수 있다.

여담으로 중요한 프로세서가 이를 사용할 수 있을 수도 있다. 그러므로 종료할 프로세서가 무엇인지 정도는 파악하고 종료하는 것이 좋다.

만약 해당 디렉토리를 사용하는 유저를 확인만 하고 싶은 경우에는 다음 명령어를 사용한다.

# fuser -cu (마운트 경로)
