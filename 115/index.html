VSCode에서 go 패키지를 못찾는 문제

2024-02-06 03:54:51

현상

VSCode(code-server)에서 go 프로젝트 진행 중, 환경변수 설정(GOPATH, GOROOT 등)을 완료하였고 go get 명령어로 패키지 설치도 하였으며, 패키지 설치 여부까지 확인을 하였으나 아래와 같은 오류가 발생할 때.

△ 패키지를 못찾아오는 모습

△ 에러 메시지

could not import golang.design/x/clipboard (cannot find package "golang.design/x/clipboard" in GOROOT or GOPATH)

원인

VSCode의 workspace 경로의 하위에 go 프로젝트가 위치하기 때문

해결

최상위에서 프로젝트를 연다.

File -> Add Folder to Workspace...

이후 들어가면 해결되어 있다.
