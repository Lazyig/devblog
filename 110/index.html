윈도우11에서 자동 업데이트 끄기

2023-06-22 14:21:11

윈도우 11에서 자동 업데이트로 인한 재부팅으로 실행하고 있던 환경이 정상 동작하지 않는 문제가 종종 발생한다.

특히 서버를 구동하고 있다면 자동 업데이트로 서버가 내려가는 것 역시 큰 문제이다.

이 경우 자동 업데이트를 비활성화여 이러한 사태를 막을 수 있다.

윈도우키 + R 을 누른 다음gpedit.msc를 입력하여 로컬 그룹 정책 편집기를 실행한다.

사용자 구성 > 관리 템플릿 > Windows 구성 요소 > Windows 업데이트 > Windows Server 업데이트 서비스에서 제공하는 업데이트 관리 > 모든 Windows 업데이트 기능을 엑세스할 수 있는 권한 제거

위 경로로 버튼을 눌러서 뜬 팝업에서 사용안함 선택 후 확인

이렇게 하면 자동 업데이트가 실행되지 않는다.
