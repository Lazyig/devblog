VMWare RaspberryPI OS Installation step failed Error

2021-11-02 22:28:28

VMWare에서 라즈베리파이 OS를 설치하던 중 다음과 같은 오류가 발생하였다.

Installation step failed
An installation step failed. You can try to run the failing item again from the menu, or skip it and choose something else. The failing step is: Install the system

여러가지 원인이 있었겠지만 당시 환경으로는

VMware Workstation 15 Player, 가상머신 램 512GB, 가상머신 설정 데비안 버전 9.0으로 설정하여 진행하였었다.

실제 라즈베리파이OS의 기반은 데비안 버전 10.0 이었으므로 이것이 원인이 아닐까 생각한다.

새롭게 시도한 것은 15 버전에서는 10.0 관련 설정이 없었으므로 16으로 업그레이드 후 데비안 버전을 수정하고 램도 2기가로 증가시켜서 재 시도하였다.

일단은 잘 성공하였으며, 해당 구축시도는 다음 포스트를 참고 (실패 과정은 생략된 포스트)
