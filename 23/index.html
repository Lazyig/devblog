윈도우 한글 자음모음 분리되서 입력되는 오류

2021-10-27 21:56:45

윈도우에서 갑자기 한글 자음 모음이 분리되서 입력되는 경우가 발생하였다.

ㅇㅜㅣㄴㄷㅗㅇㅜ ㅎㅏㄴㄱㅡㄹ ㅈㅏㅇㅡㅁㅁㅗㅇㅡㅁ ㅂㅜㄴㄹㅣㄷㅗㅣㅅㅓ ㅇㅣㅂㄹㅕㄱㄷㅗㅣㄴㅡㄴ ㅇㅗㄹㅠ

이에 대한 해결방법을 찾아서 메모해두려고 한다.

원인

원인은 Window + H키를 눌러서 윈도우 받아쓰기 기능을 활성화하면 지원하는 언어가 아니라는 메시지와 함께 발생한다.

해결방법

작업관리자를 켜서 "Microsoft Text Input Application"이라고 하는 프로세스를 종료하면 된다.

작업 관리자는 Window + R을 눌러서

다음 창에서 taskmgr을 입력하고 확인을 누르자.

아니면 시작메뉴를 오른쪽 클릭 후에 작업관리자를 눌러도 된다.
