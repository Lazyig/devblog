REMOTE HOST IDENTIFICATION HAS CHANGED! 에러

2021-10-16 23:26:42

@    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
IT IS POSSIBLE THAT SOMEONE IS DOING SOMETHING NASTY!
Someone could be eavesdropping on you right now (man-in-the-middle attack)!
...

리눅스 서버를 새로 설치 한 후 해당 서버에 깃 push를 진행하려다가 발생한 에러이다.

해당 에러는 내가 가지고 있는 인증키가 서버의 인증키와 다를 경우에 발생하며 중간자공격을 경고한다.

만약 접속하려는 서버의 인증 정보가 변경될 일이 없다면 이에 대한 의심을 반드시 하여 중간에서 누군가 서버 접속을 가로채려는 것인지 꼭 확인하고 해결방법을 실행하도록 하자.

해결 방법

$ ssh-keygen -R [도메인 or IP]

위의 명령어를 클라이언트에서 입력하여 새로운 키를 발급받으면 간단하게 해결된다.
