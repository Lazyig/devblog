Discord Installation has failed 오류 해결법

2021-10-13 21:55:00

(티스토리 블로그에서 이전하여 작성한 글 입니다.)

가끔 디스코드가 자기 멋대로 망가지는 경우가 있다.

이러한 경우 재설치를 하려 해도 다음과 같은 오류 메시지가 뜨면서 재설치가 불가능한 경우가 있다.

Installation has failed
There was an error while installing the application.
Check the setup log for more information and contact the autor.

재설치를 진행해도 오류가 생겨서 어찌할 줄 몰랐던 기억이 있습니다.

순서대로 하나씩 조치해나가시면서 재설치를 진행해보시기 바랍니다.

보통은 프로세스를 종료하는 첫번째 방법이 원인인 경우가 많습니다만 다른 원인도 있을 수 있으니 안되실 경우 추가로 진행해보시기 바랍니다.

1. 프로세스 종료

우선 작업관리자를 들어가줍니다.

윈도우 + R키로 실행을 들어간 뒤에 'taskmgr'를 적고 확인을 누르시거나,

Ctrl + Alt + Delete키를 누르신 다음 작업관리자를 눌러주면 실행이 됩니다.

그 이후에는 디스코드와 관련된 모든 프로세스들을 종료해줍시다.

1번 방법 끝.

2. 설정파일 삭제 1

디스코드 설정파일들을 삭제하는 방법입니다.

윈도우 + R키로 실행을 들어간 뒤에 아래의 주소를 적고 확인을 눌러주세요.

%USERPROFILE%\AppData\Local

해당 주소에서 Discord 폴더를 찾아서 제거해주세요.

2번 방법 끝.

3. 설정파일 삭제 2

이번엔 다른 곳에 있는 관련된 파일들 입니다.

윈도우 + R으로 실행을 들어간 뒤에 아래의 주소를 적고 확인을 눌러주세요.

%APPDATA%

다음 주소로 들어간 다음 Discord 폴더를 찾아 삭제를 해주세요.

3번 방법 끝.

마치며

위의 조치사항들을 하고 나면 정상적으로 될 것 입니다.

저는 해당 방법들로 설치를 완료하였으나, 그래도 안된다면 다른 해결방안을 찾으셔야하겠지요.

다른 유형의 오류와 해결방법을 찾으시면 공유하여 주시면 감사하겠습니다.
