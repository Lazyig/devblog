nvm node version not found 에러

2022-01-26 20:52:47

에러 발생 상황

nvm을 설치한 뒤 node를 설치하기 위해서 명령어를 입력해보았으나 설치가 동작하지 않고 해당 에러가 발생하였다.

nvm install node
nvm install 17.4.0

문제를 해결하기 위해서 해당 명령어 이후 다음 명령어로 설치할 수 있는 항목을 검색하였다.

nvm ls-remote

결과는 io-js 라는 것만 나오고 node 관련은 아에 없었다.

해결

열심히 검색을 하다가 해당 글에서 그냥 nvm을 최신버전으로 설치해봐라! 라는 글이 있었다.

마침 현재 설치한 nvm 버전은 0.33 이였고, 해당 글과 같았기에 다음 명령어로 재설치 후 해결되었다.

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
