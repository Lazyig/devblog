Gitlab 업데이트

2021-10-13 21:55:00

서론

Gitlab 업데이트를 하는 방법을 간단히 메모해놓고 발생하는 문제들을 정리해놓기 위하여 작성하게 되었다.

본문에서는 관리자 권한이 필요한 커맨드의 경우에는 명령어 앞에 #이 붙어 있으니 알아서 sudo를 붙이시던지 관리자 계정으로 들어가서 작업하기를 바란다.

사전 환경

  • OS / 배포판 : Linux / Ubuntu
  • Gitlab 정보 :
    • 옵니버스식 설치
    • 버전 : 14.3.0-ce
    • 업데이트 대상 버전 : 14.3.2-ce

본문

패키지 매니저를 통하여 옵니버스식 설치를 하였다면 이를 관리하기는 쉽다.

# apt-get install gitlab-ce

해당 명령어로 업데이트를 우선 진행해준다.
