라즈비안에 한글 설정하기

2021-11-03 00:30:21

라즈비안의 설치에 관련한 글은

/30/index.html

위의 글에서 진행하였으며, 한글 설정까지는 마친 상태이다.

1. 한글 폰트 설치

한글 설정을 마친 후 웹 브라우저를 실행해보면 한글이 깨져있다.

원인은 폰트가 없어서이다.

상단의 터미널을 열고 다음과 같이 설치를 진행하면 된다.

sudo apt-get install fonts-nanum

설치를 마치게 되면 다음과 같이 한글이 정상적으로 뜨게 된다.

2. 한글 입력기 설치

이제 한글을 입력할 수 있도록 입력기 설정을 해야할 차례이다.

sudo apt-get install im-config nabi -y

다음 명령어로 입력기 설정과 한글 입력기를 설치해주자.

im-config : 입력기 설정

nabi : 한글 입력기

그 다음 터미널에

im-config

명령어를 입력하여 입력기 설정을 실행한다.

hangle 체크 후 OK

한/영 전환은 Shift + Enter로 하면 됩니다.
