github page 언더바(_) 문제

2023-05-03 15:16:17

발단

한 동안 블로그 생성기를 잘 쓰다가 이미지가 누락되는 문제가 발생하였다.

나머지 이미지는 멀쩡한데. 해당 이미지만 누락되어 찾아보니 해당 이미지는 https://devblog.lazyig.com/static/_RYSwFejKVgF11wrBzzIz0oOv1WT204fOJ3SwSstJIU.png 이러한 경로였는데.

이는 내가 이미지를 해싱한 뒤 base64 인코딩을 해서 제공한 이름이다. 마침 _로 시작하여 찾아보니 깃허브 페이지에서는 언더바가 안먹힌다고 한다...

원인

Github Page에서는 _로 시작하는 경로나 파일을 무시한다.

해결

혹시나 이미지나 페이지 혹은 특정 경로 하위가 안먹히는 오류가 있을 경우 언더바를 지우도록 하자.
